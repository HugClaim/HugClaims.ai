# Persona Folder

## Files
- `Persona.md`: structured, stable persona profile used for task personalization.
- `Personal_Wiki.md`: human-readable "Wikipedia-like" profile page.
- `evidence/`: dated evidence notes extracted from interactions.
- `snapshots/`: periodic persona snapshots (e.g., `Persona_2026-06-01.md`).

## Update Process
1. Add a dated note under `evidence/` from new interactions.
2. Promote repeated signals into `Persona.md`.
3. Refresh `Personal_Wiki.md` narrative when material changes.
4. Save periodic snapshot in `snapshots/`.
